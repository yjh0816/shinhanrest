# shinhanrest
This is djangoframework practice that use Django REST Framework with ShinhanInvestment. shinhanrest is RESTful API

## Installation
```shell
$ pip install django
$ pip install djangorestframework
```
## How to start
```shell
$ python manage.py runserver
```

## Features
- Product read API
- Product write API
